## Codebase

### Variables:
#### Variable: default_ai_prompt

Objectif: Cette variable est une liste de chaînes de caractères. Chaque chaîne de caractères est une invite qui peut être utilisée pour guider l'IA dans la génération de texte. Ces invites sont utilisées par défaut si aucune autre invite n'est fournie.

La liste contient les éléments suivants :

1. "Vous êtes un ingénieur logiciel et un architecte logiciel hautement qualifié"
2. "Vous analysez le code d'une autre personne et rédigez un rapport sur chaque fichier dans une base de code"
3. "Veuillez donner un compte rendu détaillé de la manière dont chaque importation, classe, méthode, décorateur et variable importante fonctionne dans le code et de son intention"
4. "Ces sections sont séparées par ***"
5. "Présentez tout de manière à ce qu'un nouveau développeur puisse vraiment comprendre ce que le code est censé faire"
6. Un exemple de format Markdown pour documenter un fichier de code.

Ces invites sont utilisées pour guider l'IA dans la génération de texte. Par exemple, la première invite pourrait être utilisée pour générer du texte qui décrit les compétences et les responsabilités d'un ingénieur logiciel et d'un architecte logiciel. La deuxième invite pourrait être utilisée pour générer du texte qui décrit le processus d'analyse du code et de rédaction d'un rapport. Et ainsi de suite.
